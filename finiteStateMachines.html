A finite state machine consists of a finite amount of states and a current state. The device uses the input to transition from one state to the other until it reaches the accepting state (if the input is accepted). Mealy machines are a type of state machine which produce outputs.

State transition diagram

A state transition diagram is one way of representing a FSM. Circles represent states (labeled with its name 'S01' etc), Lines represent a transition (labeled with the condition of the transition) and double-circles represents an accepting state.

FSM representing a vending machine which requires £1 in diagram form.

State transition table

A state transition table expresses a FSM in tabula form. Input on the left, current state in the middle and next state on the right.

Input Current State Next State
20 S0 S20
50 S0 S50

Mealy Machines

Mealy machines are a subset of FSM's that produce output. This output can be represented after a '/' on a trasition on a state transition diagram, or another column on a state transition table.
